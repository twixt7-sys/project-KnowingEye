Database Design & Persistence

Database Strategy

AspectSpecification
Production RDBMSPostgreSQL 15 or newer
Development RDBMSSQLite (zero-config local dev)
ORMDjango ORM with migrations
Namingsnake_case table and column names
Primary keysAutoField (integer) for User, Exam, Question, Response, SessionLog; UUID for ExamSession, BehaviorLog, Alert, SessionIdentityReference
JSON fieldsQuestion options, session log details, behavior metadata, identity embeddings

Entity Summary

TableEntityDescription
authentication_userUserAccounts with ADMIN or EXAMINEE role, profile fields, avatar
exams_examExamExam definition: duration, passing score, scheduling, status
exams_questionQuestionQuestions with type, options JSON, correct answer, points
user_sessions_exam_sessionExamSessionOne attempt; UUID PK; tracks timer, score, IP, status
user_sessions_responseResponsePer-question answer with auto-grading for MCQ/TF
user_sessions_session_logSessionLogAudit events: started, submitted, terminated, expired
behavior_logBehaviorLogAI-detected behavioral events with score and confidence
behavior_alertAlertFlagged suspicious events with severity and resolution state
monitoring_identity_referenceSessionIdentityReferenceOne enrolled face embedding per session (OneToOne)

Relationships

  • User 1-* Exam (as creator via created_by)
  • Exam 1-* Question
  • User 1-* ExamSession; Exam 1-* ExamSession
  • ExamSession 1-* Response; Question 1-* Response
  • ExamSession 1-* SessionLog, BehaviorLog, Alert
  • ExamSession 1-1 SessionIdentityReference

Constraint: at most one in_progress session per (exam, user) pair.

Behavior Event Types (enum)

no_facemultiple_faceslooking_awaybad_postureleaving_seatobject_detectedidentity_mismatchsuspicious_pattern

Indexes (Performance)

  • User: role, email, username (unique)
  • Exam: status + created_at, created_by, exam_code, availability window
  • Question: (exam_id, order) unique
  • ExamSession: (exam, user), (status, started_at), (user, started_at)
  • BehaviorLog / Alert: (session, timestamp/created_at)
