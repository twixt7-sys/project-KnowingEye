Chapter I

Definition of Terms

Draft terms from the outline and codebase. Review for completeness (~10% per WBS).

  • Online examination platform — system used to create, administer, and score exams digitally.
  • Computer vision — techniques enabling computers to interpret visual data from images or video.
  • Behavioral monitoring — analysis of examinee actions such as gaze, posture, and facial movement during a session.
  • Deep learning — neural network methods for pattern recognition and classification.
  • ArcFace — deep face-recognition model producing embeddings for identity verification.
  • YOLO — real-time object-detection family used for person/object detection.
  • Session — a single timed exam attempt tracked from start to submission.
  • WebSocket — full-duplex protocol for streaming frames and alerts in real time.
