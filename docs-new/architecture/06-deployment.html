Deployment Architecture

2.1.4.1 Web-Based Environment

The system is deployed as a browser-accessible web application. Examinees and administrators use HTTPS; no native app store distribution. Webcam access requires browser permission (MediaDevices API) and secure context (HTTPS in production).

Deployment Steps (Summary)

  1. Provision PostgreSQL and Redis
  2. Configure backend/.env
  3. pip install -r requirements.txt + ML deps
  4. python manage.py migrate && collectstatic
  5. Run Daphne behind systemd/supervisor
  6. npm run build → deploy SPA to Nginx root
  7. Configure Nginx for /api/, /ws/, static, media
  8. Smoke test: /api/monitoring/health/

Full guide: docs/deployment.md
