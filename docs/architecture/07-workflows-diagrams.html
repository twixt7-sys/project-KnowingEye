System Workflows & Diagram Index

Sequence and data-flow diagrams as machine-readable JSON. Use placeholders below in your paper; render figures from the linked JSON/DBML files.

Examinee Exam Workflow

  1. Login → obtain JWT
  2. Dashboard → select published exam
  3. Start session → receive questions + session UUID
  4. Open monitoring WebSocket → enroll reference face
  5. Answer questions (parallel monitoring loop)
  6. Submit session → receive score and pass/fail
  7. View results / profile

Administrator Workflow

  1. Login as ADMIN
  2. Create exam and questions → publish
  3. Monitor live sessions via dashboard + WebSocket alert feed
  4. Inspect session behavior logs and resolve alerts
  5. Generate reports and export CSV
