System Scope Summary

Included

  • Web-based exam management and delivery (desktop/laptop + webcam)
  • Role-based access: Administrator (examiner) and Examinee
  • Session-guided exam taking with timer and answer tracking
  • Real-time behavior monitoring via computer vision (face, gaze, posture, objects)
  • Behavior logs, alerts, and administrative reporting

Excluded (per project scope)

  • Native mobile application
  • External institutional SIS integration
  • Specialized biometric hardware
  • Nationwide / multi-region deployment at this stage
